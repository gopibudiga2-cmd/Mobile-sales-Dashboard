# Mobile-sales-Dashboard
🚀 Project Overview

This project presents an interactive Mobile Sales Dashboard built using Power BI to analyze sales performance across different dimensions such as city, brand, time, and payment methods.

The dashboard provides meaningful insights into sales trends, customer behavior, and transaction patterns, helping businesses make data-driven decisions.

📌 Key Features
📍 City-wise Sales Analysis
Visual map showing total sales distribution across different locations in India.
📈 Time-based Insights
Monthly and daily trends (MTD analysis)
Year-over-Year (YoY) and Same Period Last Year comparison
📦 Brand Performance
Sales comparison across top brands (Apple, Samsung, OnePlus, etc.)
💳 Payment Method Analysis
Distribution of transactions via UPI, Debit Card, Credit Card, and Cash
⭐ Customer Ratings
Breakdown of customer feedback (Good, Average, Poor)
🔄 Interactive Filters
Mobile Model
Payment Method
Brand
Year / Month selection
